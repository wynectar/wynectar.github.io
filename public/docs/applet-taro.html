项目开发步骤 ​

温馨提示

  • Taro 项目基于 node，请确保已具备较新的 node 环境（>=16.20.0）。
  • 由于 Taro 部分能力使用 Rust 开发，在 Windows 上，请确保安装了 Microsoft Visual C++ Redistributable。点击查看
  • 在 macOS 或类 Unix 系统中，当你遇到 permission denied 错误时，通常是因为你没有足够的权限来执行某个操作：sudo chown -R [用户名] [文件路径]，例如 sudo chown -R admin /Users/admin/Desktop/taro-applet/

1. 项目初始化 ​

方式一 ​

  1. 需要使用 npm 或者 yarn 全局安装 @tarojs/cli
bash 
npm install -g @tarojs/cli
1
bash 
yarn global add @tarojs/cli
1
bash 
pnpm install -g @tarojs/cli
1

温馨提示

  • 因权限问题造成的安装失败，需要在命令前面加 sudo
  • 可以使用 npm info 查看 Taro 版本信息：npm info @tarojs/cli

Taro 版本信息截图

  1. 初始化
bash 
taro init myApp
1

初始化时根据自己的需求进行选择，建议选择CLI 内置默认模版，因为其他模版可能会有demo报错的情况。 初始化模版选择截图

方式二 ​

npm 5.2+ 也可在不全局安装的情况下使用 npx 创建模板项目，初始化如下：

bash 
npx @tarojs/cli init myApp
1

2. 安装UI库和依赖 ​

如果不使用 taro-ui 和 微信工具 可以跳过此步骤，如果需要开发其他小程序，需要装对应的工具。

bash 
npm install taro-ui @tarojs/plugin-platform-weapp
1

yarn add taro-ui @tarojs/plugin-platform-weapp
1
bash 
pnpm install taro-ui @tarojs/plugin-platform-weapp
1

3. 下载小程序开发者工具 ​

  • 微信开发者工具
  • 百度开发者工具
  • 支付宝小程序开发者工具
  • 抖音小程序开发者工具
  • QQ 小程序开发者工具
  • 京东小程序开发者工具

需要注意开发者工具的项目设置：

  • 需要关闭 ES6 转 ES5 功能，开启可能报错
  • 需要关闭上传代码时样式自动补全，开启可能报错
  • 需要关闭代码压缩上传，开启可能报错

4. 启动项目: weapp ​

以微信小程序为例，编译命令如下：

bash 
# pnpm
$ pnpm dev:weapp
$ pnpm build:weapp

# yarn
$ yarn dev:weapp
$ yarn build:weapp

# npm script
$ npm run dev:weapp
$ npm run build:weapp

# 仅限全局安装
$ taro build --type weapp --watch
$ taro build --type weapp

# npx 用户也可以使用
$ npx taro build --type weapp --watch
$ npx taro build --type weapp

# watch 同时开启压缩
$ set NODE_ENV=production && taro build --type weapp --watch # CMD
$ NODE_ENV=production taro build --type weapp --watch # Bash

5. 开发调试: weapp ​

  1. 在微信开发者工具中导入项目

  2. 选择项目目录 dist 文件夹

  3. 开启真机调试功能

  4. 使用预览功能测试

导入 dist 文件夹后，就已经可以调试了；但你如果想使用真机调试，需要配置AppID(小程序ID)，如下：

json 
{
"miniprogramRoot": "./dist",
"projectname": "taro-applet",
"description": "基于taro的小程序模版",
"appid": "touristappid",  
"appid": "AppID",  
"setting": {
    "urlCheck": true,
    "es6": false,
    "enhance": false,
    "compileHotReLoad": false,
    "postcss": false,
    "minified": false
},
"compileType": "miniprogram"
}

小程序发布流程 ​

  1. 获取AppID(小程序ID) ：进入微信公众平台，有账户登陆查询，无账号注册获取

微信公众平台截图

  1. 把 project.config.json 中的 touristappid 替换成你自己的 AppID

  2. 确认真机调试是否有问题

  3. 点击上传，添加版本信息

上传信息截图截图

  1. 保证自己的小程序信息填写完整

  2. 小程序：管理 -> 版本管理中 -> 选择提交审核 -> 提交审核通过后选择发布

如果微信小程序未认证，则不能 被搜索 和 被分享，只能通过下载小程序码扫码查看。
